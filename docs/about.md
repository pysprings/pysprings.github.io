# About PySprings

PySprings is the **Southern Colorado Python User Group**, based in Colorado Springs. We've been meeting since 2013.

## Meetings

We meet twice a month:

- **2nd Tuesday** — Virtual meeting
- **4th Tuesday** — In-person at [East Library](https://ppld.org/locations/east-library) (5550 N. Union Blvd), Pikes Peak Library District

Check our [Meetup page](https://www.meetup.com/pysprings/) for the latest schedule and topics.

## Chat

Join us on Element/Matrix — no install required, opens in your browser:

<https://app.element.io/#/room/#pysprings_community:gitter.im>

Or use any Matrix client: `#pysprings_community:gitter.im`

## GitHub

We publish projects, lightning talks, and learning materials at <https://github.com/pysprings>.

See the [project index](wiki/projects/index.md) for a full list of our 40+ repositories.

## History

PySprings was founded in 2013. We previously offered a **Python For Absolute Beginners** class with the [Pikes Peak Library District](https://ppld.org/) (pre-pandemic). There's no current plan for bringing this back, but our [Skill Paths](paths/index.md) cover similar ground for self-study.

## Python Installation

- **Windows**: [Windows Python Setup](https://github.com/pysprings/windows-python-setup) — our presentation on installing Python with Conda
- **Linux/Mac**: Python is usually pre-installed. If not, use your system package manager or [python.org](https://www.python.org/downloads/).
