## Overview

A Django web application for tracking member "merit badges" — skills and accomplishments within the PySprings community. This was the group's most ambitious collaborative project, featuring CI, pre-commit hooks, and 16 open issues.

## Tech Stack

- Django
- Pre-commit hooks
- CI pipeline

## Historical Note

Development was most active in 2018–2019. The project is no longer maintained but remains a good example of collaborative Django development in a user group setting.
